# Мультиагентная система для генерации и проверки математических задач.
Курсовой проект первого курса магистратуры ВШЭ "Искусственный интеллект". Выполняется студентами *Проскуриным Александром* и *Гришиным Лаврентием* под руководством куратора *Артёма Беляева*.

В ходе проекта необходимо разработать и протестировать мультиагентную систему, в которой несколько LLM-агентов взаимодействуют для:
* Генерации математических задач (алгебра, геометрия, мат. анализ, теория вероятностей, школьные задачи).
* Автономного решения этих задач.
* Проверки решений и выявления ошибок.

Система будет работать через API LLM, где агенты взаимодействуют по чёткому протоколу, имитируя работу “педагогической команды”. Дополнительно возможно интегрировать `SymPy` или `Wolfram Alpha` через function calling для проверки и расчётов.

Подробный план проекта, таймлайн его выполнения и финальные результаты отражены в [Google Doc](https://docs.google.com/document/d/1_VHg8ExH8BJVLfXHY7-XOwSwFmaxtwgonmlgO18QJ7Y/edit?tab=t.0).
